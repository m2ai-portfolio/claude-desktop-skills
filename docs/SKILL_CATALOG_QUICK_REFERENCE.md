## USAGE TIPS

### How Skills Get Triggered
Claude automatically selects skills based on your request. You don't need to name the skill - just describe what you need:

✅ "Help me write resume bullets for my data analyst role"  
✅ "Create an Instagram caption for my fitness post"  
✅ "Summarize this meeting transcript with action items"

### Best Practices
1. **Be specific** - More context = better output
2. **Iterate** - Ask for adjustments if needed
3. **Combine skills** - Use multiple in sequence
4. **Save examples** - Document what works well

### Project Organization

**Option 1: Separate Projects by Suite**
- Project: "M2AI Career Tools" (6 skills)

- Project: "M2AI Growth Tools" (5 skills)
